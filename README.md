# social-app-95

love posting about stuff, hate the rounded borders of the modern web, this is the app for you

steps from root

`1:`

```
cd client/
```

`2:`

```
yarn add
```

or

```
yarn install
```

or

```
yarn
```

`3:`

```
cd ../server/
```

`4:`

```
npm i
```

or

```
npm install
```

`5:`

```
npx knex migrate:latest
```

`6:`(optional)

```
npx knex seed:run
```

`7:`

```
cd ../
```

`8:`

```
npm i
```

or

```
npm install
```

`9:`

```
npm run dev
```
